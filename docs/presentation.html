09 RAG Pipeline — Hybrid Search

User Query
1. Embed Query → all-MiniLM-L6-v2 → 384-dim vector
2. Build Filter: is_active + price range + city + brand + ...
3. Hybrid Search — RRF Fusion
BM25 Full-texttext named vector
Vector Cosinedefault named vector
4. Quality Evaluation: top_score > 0.65 AND avg_score > 0.5?
✅ Yes → Return results
🔄 No → Broader retry

12 Data Flow: User → PG → Embedding → Qdrant

✏️ Write Path — Ad Creation
User submits ad form
POST /api/v1/ads → Nginx → Backend
Save to PostgreSQLads table + storage
Background: Index to Qdrant
a. Build embedding text: "BMW 520 2022 sedan ..."
b. Embed → all-MiniLM-L6-v2 → 384-dim vector
c. Build payload (15 metadata fields) d. Upsert to Qdrant
Set qdrant_synced = true
📖 Read Path — Search
User types: "cheap reliable sedan Cairo under 500k"
GET /api/v1/search?q=...&price_max=500000&city=Cairo
1. Embed query → 384-dim
2. Build filter (city+price)
3. Hybrid search: BM25 + Vector → RRF fusion
4. Quality eval → 5. Enrich from PG → Return to user
